*****
Usage
*****

Solution design prioritizes being environment agnostic, meaning most solutions can be run anywhere given the minimum requirements are met. Check on the solution for the minimum requirements.

If you use the F5 provided environment, there are two ways to access it, a course and a blueprint. Typically courses are used for external to F5 employees, and blueprints are used internally.
